## THE SOLUTION

### **How roostr Works:**

**18 AI Agents Hunt Stocks Daily:**
- 3 roostr custom agents (Social Arbitrage, Value Investing, Catalyst)
- 12 legendary investor agents (Buffett, Munger, Burry, Damodaran, etc.)
- 3 quantitative agents (Fundamentals, Sentiment, Technicals)

**Ensemble Intelligence:**
- All agents analyze 100+ stocks daily
- Agents suggest top 5-10 opportunities (highest conviction)
- Agents debate ‚Üí vote ‚Üí generate conviction document
- Conviction threshold: >7/10 to deploy (discipline > hype)

**Human Discipline:**
- Joselo (Risk Manager) validates every trade
- Enforces stops (-8% to -10%, no exceptions)
- Can REJECT any trade (even if all agents agree)
- Max portfolio loss: 3.8% (risk caps enforced)

**Full Transparency:**
- Every decision documented publicly (GitHub: roostr-research)
- Live dashboard (real-time portfolio, P&L, positions)
- Daily Twitter updates (wins AND losses)
- Conviction docs (full agent reasoning published)
