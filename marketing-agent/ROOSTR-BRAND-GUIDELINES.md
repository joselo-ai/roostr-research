### **Typography**

**Primary Font (Code/Data):**
- **Family:** Monaco, Menlo, Courier New (monospace)
- **Usage:** Dashboards, data displays, technical content
- **Weights:** Regular, Bold
- **Why:** Terminal aesthetic, tech credibility, readability for numbers

**Secondary Font (Marketing):**
- **Family:** Inter, SF Pro, Helvetica Neue (sans-serif)
- **Usage:** Marketing copy, social media, long-form content
- **Weights:** Regular, Medium, Bold
- **Why:** Clean, modern, accessible, web-friendly

**Font Hierarchy:**
```
H1: 2.5em Bold (marketing headlines)
H2: 1.8em Bold (section headers)
H3: 1.3em Medium (subsections)
Body: 1em Regular (primary text)
Code: 1em Monospace (data, metrics)
Small: 0.9em Regular (captions, metadata)
```

---

### **Visual Style**

**Design Principles:**
1. **Dark Mode Native** - No light mode. Ever. Dark = professional, focus, terminal.
2. **Data-First** - Charts, dashboards, numbers front and center
3. **Minimal Clutter** - Every element serves a purpose
4. **Terminal Aesthetic** - Inspired by command-line interfaces, hacker terminals
5. **Bold Contrast** - Electric green on dark navy = high visibility

**UI Elements:**
- Rounded corners: 8px (modern, friendly)
- Border: 1px solid rgba(255,255,255,0.1) (subtle separation)
- Shadow: None or minimal (flat design)
- Backdrop blur: 10px (glass morphism on panels)
- Gradients: Subtle (135deg, navy to slightly lighter navy)

**Imagery Style:**
- Charts: Line charts, candlesticks, clean data viz
- Icons: Geometric, minimal, monochrome
- Illustrations: Rare. Only when adding value (rooster mascot)
- Photos: None. Data > faces.
