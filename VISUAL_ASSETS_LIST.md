### Icon Download Instructions

**Method 1: SVG (Figma, high-quality)**
1. Go to https://lucide.dev
2. Search icon name
3. Click "Copy SVG"
4. Paste into Figma (Cmd+V / Ctrl+V)
5. Resize to desired size (24Ã—24, 32Ã—32, etc.)
6. Change stroke color (Black or Green)

**Method 2: PNG (Google Slides, quick)**
1. Go to https://lucide.dev
2. Search icon name
3. Click "Download PNG"
4. Choose size (64px, 128px, 256px)
5. Import into Google Slides (Insert â†’ Image)
6. Resize as needed
7. Recolor using "Format options â†’ Recolor"

**Method 3: Icon Font (Keynote, PowerPoint)**
1. Download Lucide icon font: https://github.com/lucide-icons/lucide/releases
2. Install font on system
3. Use character map to insert icons as text
4. Format as text (size, color)

## 2. Charts & Graphs

**Style:**
- Minimal chrome (no gridlines, borders, shadows)
- High data-ink ratio (maximize data, minimize decoration)
- Single color (Green) for primary data
- Black for comparison/baseline data
- Direct labels (no legends when possible)

**Color:**
- Primary: Green (#4ade80)
- Comparison: Black (#000000)
- Negative/Risk: Red (#ef4444) â€” use sparingly

## Final Notes

**Philosophy:** Less is more. Every visual element should serve a purpose (data, emphasis, navigation). Avoid decoration for decoration's sake.

**Consistency:** Use the same icon style (Lucide), same colors (black, white, green), same stroke weight (2px) across all assets.

**Accessibility:** Ensure all charts/diagrams are readable at small sizes (Zoom test: view at 50% zoom, still legible?).

**Backup:** Save all assets in multiple locations (local + cloud). Use version control (Git) for source files.
